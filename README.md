# 🌦️ Forecasting Weather Yogyakarta

### AI-Powered Meteorological Prediction

Sistem prediksi cuaca berbasis **Deep Learning (LSTM)** yang dirancang untuk memantau kondisi atmosfer di wilayah Yogyakarta dan sekitarnya.

---

### 📊 Model Statistics
- **Architecture**: Multi-layer LSTM (Long Short-Term Memory)
- **Hardware**: NVIDIA GPU Accelerated (CUDA)
- **Horizon**: 60-Day Future Forecast
- **Parameters**: Temp, Humidity, Rain, Pressure (hPa)
